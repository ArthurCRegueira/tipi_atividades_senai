Exercício 01

Crie um programa que leia 10 números e mostre a soma.
Usando while(){}
var noel = 1;
var soma = 1;
while(noel < 10){
var numeros = prompt("digite um número: ");
soma += parseFloat(numeros);
noel += 1;
}

Exercício 02

Crie um programa que leia 10 números e mostre a soma.
Usando do{}while();
var noel = 1;
var soma = 1;
do{
 var numeros = prompt("digite um número: ");
 soma += parseFloat(numeros);
 noel += 1;
}while(noel < 10);
document.write("A soma é ", soma);

Exercício 03

Crie um programa que leia 5 números e mostre o maior.
Usando For(){}
var maior = prompt("Digite um número para comparar");
for (let index = 0; index < 4; index++) {
  input = prompt("Digite um número para comparar: ")
  maior = input>maior? input : maior;
} document.write( O maior número é" , maior);
